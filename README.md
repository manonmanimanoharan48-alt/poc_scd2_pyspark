# poc_scd2_pyspark
implementing SCD type 2 with Pyspark using Microsoft notebook in Fabric.

1. Organize data with medallion architecture.
2. Ingest CSV files into Lakehouse using copy job.
3. Apply data transformation/cleaning on tables using  Pyspark notebook.
4. Apply SCD2 type logic using Pyspark notebook .
5. Run/monitor notebook using pipelines and event triggers.
